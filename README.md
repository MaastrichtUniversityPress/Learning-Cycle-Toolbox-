# Learning Cycle Toolbox

This Learning Cycle Toolbox is designed to support the learning process and the development of self regulated learning (SRL) in higher education. It is intended for both students and teachers who guide students in strengthening their SRL skills. 
The tools align with the different phases and steps of the (self regulated) learning cycle and can independently be used by students or initiated by the teacher. In addition, tools offer structure for student team meetings and one on one coaching sessions. All tools are brief, practical, and easy to apply — no long texts.
Developed by the Bachelor of Medicine team and published by Maastricht University Press, the toolbox was created for UM’s medical curriculum. However, it may also be useful (as an inspiration) for other higher education contexts where self regulated learning is encouraged.
How to use this toolbox: </br>
•	Go to the Learning Cycle Toolbox start page.</br>
•	From there, navigate to the tools that correspond to the SRL phase, step, or skillset you want to work on.
